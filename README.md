# Tertiary Infotech — WSQ Courseware Design Skills

Claude Code skills for producing **Tertiary Infotech Academy** WSQ courseware in the house style (white theme, Arial, blue `#1F6FEB` / teal `#10B981`).

| Skill | Purpose |
|-------|---------|
| **tertiary-course-slides** | Generate a 3-day WSQ course slide deck (`python-pptx`) — cover, card/icon **admin pages** (blank + filled *About the Trainer*, Digital Attendance/TRAQOM, Ground Rules, Lesson Plan, Learning Outcomes, **Assessment Flow**, **Certification & TRAQOM**), n8n key-concept slides, and per-activity overview/screenshot/step slides. Bundled white-glyph icons in `assets/icons/`. |
| **tertiary-ppt-design** | House-style design system & rules for professional WSQ decks. |
| **tertiary-learner-guide** | WSQ Learner Guide as aligned Markdown + Word (.docx) with WSQ cover, version-control record, TOC, footer. |
| **tertiary-lesson-plan** | WSQ Lesson Plan (.docx) with house cover, colour-coded daily schedule (8 instructional hours/day). |

## Install
Copy a skill folder into `~/.claude/skills/`, then invoke it from Claude Code.
